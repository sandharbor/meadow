LaTeX Math

Inline math: E=mc2E = mc^2

Block math:

−b±b2−4ac2a\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}

LaTeX inside code blocks should not be transformed: 

$E = mc^2$ and $$\frac{1}{2}$$

Inline code too: $E = mc^2$
