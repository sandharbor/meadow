[[app component]]

If you are in the [[app component - modal -- create or edit site]] and the title you have chosen is one of the [[source pages with duplicate titles]], then this tooling shows you all the candidates, with the _link not tracked_, so you can pick the right one.

We don't force the user to specify the _link not tracked_ in the [[app component - modal -- create or edit site]] as an [[affordance]].  Most of the time the [[site page config -- title]] is enough... duplicates are uncommon.
