This is a special mode where you provide a [[source page]] , either by using the [[Meadow Obsidian plugin]] or by clicking the "find in sites" action in the [[app component -- site page card]] dropdown

It opens the [[app page -- sites list]] and shows all the [[site]]s containing the page.

It also allows allows you to easily [[create a new site from a source page]]

Related to [[multi site management]].
