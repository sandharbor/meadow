^ orphaned [[site page tracking state -- tracked]]

---

You can have a [[site page]] that was in [[site page tracking state -- tracked]], but then, because of a [[source graph change]], when you go back to the site in [[site lifecycle -- ongoing updates]], the link to it can be removed, so It is no longer pointed to by any [[site page path]].  So, it is orphaned.

Ultimately we will need to deal with that in [[orphaned tracked site page resolution tooling]]
