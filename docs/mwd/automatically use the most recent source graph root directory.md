If you are not using the [[Meadow Obsidian plugin]], then when you [[create a new site from a source page]], you need to supply both the [[site page type -- initial]] and the [[source graph root directory]].

That's a bit of a pain, but it is necessary when you are [[doing things the first time]].

But from the second time onwards we can provide an [[affordance]] where we automatically use the most recent [[source graph root directory]].  We do this to help with [[design motivation -- speed]], and with [[design motivation -- easy setup]], since it means that someone can use the app pretty effectively without needing to install the [[Meadow Obsidian plugin]]
