Take a [[raw working graph]] and set [[site page tracking state -- tracked]] on some (or all) of the site pages.  The tracked working graph is those pages.

The [[final working graph]] is only the [[whitelist]]ed [[site page]]s in the tracked working graph.

The tracked working graph does not contain [[frontier site page]]s because [[frontier site pages cannot be tracked]]

The tracked working graph is not necessarily a [[fully connected graph]]. It can have an [[inconsistent state -- untracked site page within controlled zone]], which we warn people about.
