Roughly the zone dictated by [[site page config -- outlinksDepth]] and [[site page config -- inlinksDepth]] constraints. Essentially all the connected site pages within the [[raw working graph]], excluding any [[frontier site page]]s, if we happen to be showing them.

This idea does not care about the [[site page tracking state]] for the [[site page]]s in it.  In other words, untracked pages can be part of the controlled zone.  Also, a [[blacklisted site page]] is part of the [[working graph controlled zone]].

]It doesn't include frontier site pages because [[frontier site pages cannot be tracked]].
