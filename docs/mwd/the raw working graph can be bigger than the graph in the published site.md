The [[raw working graph]] contains all the data required to show the [[app component -- site page views]], which can be more than is required to render the final [[published site type -- local html]].  We provide two examples of why below
:
First example: to support [[site pages filter option -- blacklisted]], _link not tracked_ but obviously we won't publish them.
:
Second example: if we show the [[frontier]], then the [[raw working graph]] shows the [[frontier site page]]s, but since [[frontier site pages cannot be tracked]], then they can never be published.
